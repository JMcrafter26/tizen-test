# Tizen Test App

This directory contains the static configuration files for the Tizen web app. It serves as the base for the final `.wgt` package.

It's just a simple test project to experiment with building Tizen apps via SvelteKit and DaisyUI.

It fetches media from the public domain [Wikimedia API](https://commons.wikimedia.org/w/api.php).

> [!IMPORTANT]
> Don't expect updates and a smooth experience, because like I said, it's just an experiment

![Screenshot](https://files.catbox.moe/qufva6.png)

## If you want to try it out

If you'd like to install the compiled app on your Tizen TV, check out [TizenBrew](https://github.com/reisxd/TizenBrew) for an easy installation process.
